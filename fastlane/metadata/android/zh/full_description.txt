提取 .zip Github/Gitlab/Bitbucket 仓库，并在移动设备上使用代码编辑器和 Markdown 查看器打开文件，适用于学生、代码阅读者和快速访问。

探索任何 Github、Gitlab 或 Bitbucket 仓库 – 只需将其下载为 .zip 文件，即可在精美的只读代码编辑器中立即打开。无需克隆。导入后无需互联网。注重隐私。

非常适合：
- 在路上学习开源项目

- 快速检查作为 zip 下载共享的仓库
- 无需设置 git 即可从著名项目中学习

- 跨文件搜索和文件内容内搜索
- 可自定义：字体大小、字体家族、缩放、亮/暗主题
- 通过 插件 系统可切换文件浏览器和高级功能
- 支持大型项目（提取过程中显示进度指示器）
